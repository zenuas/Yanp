## head集合の作成

[nullable集合](15.nullable集合の作成.md)を元に規則の先頭に来うる記号を集約する。これをhead集合とする。
非終端記号の場合は、非終端記号のhead集合を集約する。

```
start : a b
      | c
a : 'A'
b : 'B'
c : 'C'

↓

startのhead集合は a、c、'A'、'C' となる
aのhead集合は 'A' となる
bのhead集合は 'B' となる
cのhead集合は 'C' となる
```

head集合の作成は次の疑似コードで表す。
これをなくなるまで全ての行に対し行う。

```
head_first = 行頭を抽出

foreach (x in head_first)
    if (xが終端記号であれば)
        head集合[xの規則]にxを追加

foreach (x in head_first)
    if (xが非終端記号であれば)
        foreach (y in head_first[x])
            head_first[x]にyを追加
```
